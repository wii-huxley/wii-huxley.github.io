手相-智慧线

发表于 2017-08-16 | 分类于 星座运势

  • 智慧线（理智线）：和生命线大约起自同一部位，朝横的方向或稍微斜下的长线。

  • 智慧线，不仅表现知能的高低，同时表现推理力和直感力等头脑活动的优劣。

  • 智慧线明朗深长的人：表示对事物具有全力以赴的精神。虽然视其努力的程度也能获得名誉和财运，但若是大意的话，有时也会陷入人家的圈套中。

  • 智慧线太短的人:主才能拙劣，总立在社会最下层，过着贫困无依的曰子。

  • 智慧线太长的人:表示一切都以自我为中心来加以思考、行动。因为过于自信，所以难免会招来失败。

  • 智慧线紊乱的人:思绪复杂，处世缺乏规律性和组织性，经常被忧愁困扰。

  • 与生命线很接近，起自其上部的智慧线:主充满信心，且具有抵制力，只要勇敢进取，必能获得成功。

事业线又称命运线，由手?至中指之纵线，表示一个人在社会活动，事业发展及人生运程的状态。

  • 从中指下方延伸到感情线

    前运佳而晚运停滞，一般中老年时期不如少壮时期运势好。

    这种人通常在年轻时将工作视为重心并持续努力，但很少成功或者位高无权，尤其在人际关系常常失衡。

  • 延伸至与感情线重叠

    贵人运大好，年轻时多注意人际交往，中年后行大运。

  • 从月丘延伸到土星丘

    这种人面对困难有信心，长辈运很好，懂得如何照顾自己。

    如事业线好，则人缘佳，协调力强；如智慧线好，则懂得付出。如延伸到食指与中指之间，则晚运佳。

  • 从生命线延伸出感情线

    少年时期生活比较艰苦，大部分人最终能成大器。

    延伸出感情线并有分叉，金钱运尚可有做领导的实力。若止于智慧线，会因自负而判断错误，中年后更明显。

  • 手腕上方延伸至中指

    竞争意识强烈，自尊心很强，通常可以掌握权力。

    若有向小指延伸的支线，头脑好有创意，出国发展不错。若感情线附近有岛纹，在人际或爱情上会有挫折。

  • 延伸至中指分出两支线

    事业有长足发展，常常在企业中身居要职，不欠缺机遇和贵人运。

    通常有此线者在事业上可以有长足的发展，在公司常常能位居要职，如感情线和智慧线理想，前途更为光明。

  • 在金星丘出现横切线

    表示当前运势有阻碍，或遭遇小人。

    表示当前运势有阻碍，或遭遇小人，但困难不会持续很久，要督促自己不能灰心，丧失斗志。

  • 与生命线相邻延至土星丘

    年轻时生活较困难理想难实现，但中年运势转好。

    感情线附近有支线：会有多份职业或牟利方法，晚年运佳。延至太阳丘：有奉献精神，能培养出优秀人才。

  • 事业线不明显且弯曲

    这种人不容易安定下来，家庭生活也不够和谐愉悦。

    事业线断裂有岛纹，灾祸来源指示，需防备。有支线延伸至月丘：即使有灾祸也能化险为夷。
