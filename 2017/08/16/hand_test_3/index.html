手相-生命线

发表于 2017-08-16 | 分类于 星座运势

  • 生命线是食指与拇指之间，以半圆形包围拇指球的线。

  • 生命线，不仅是寿命长短、健康状态、有无病难及突发事故的部位，同时也是做预防灾厄和提高的部位。

  • 生命线细长、深秀明朗、不中断、且呈现淡红色的人：主健康良好、长命百岁。但若是凭着身体健康而暴食暴饮的人，也会导致短命。

  • 生命线短的人:主个性正直脚踏实地，但流于胆小。因体质较弱，可能多病，但不等于短，通常都会过着不满和不安的人生。

  • 生命线曲折的人:主住居经常变动、生活不安定。多半具有远渡重洋的机会。

  • 生命线分有支线的人:表示一家会分成两个家生活，也是持有两个家庭之意。

  • 生命线的末端起叉纹的人:就是“要特别注意起居生活”的一种警言。或者表示着即将离开祖国而到国外定居之意。

生命线是食指与拇指之间，以半圆形包围拇指球的线，预示个人体力强弱及健康运势。

  • 在中指中心线上弯曲并延伸

    心脏及消化系统健康，避免年轻时的暴饮暴食。

    心脏及消化系统健康，身体强壮，如第一火星丘过于发达，要避免年轻时的暴饮暴食或沉溺于某件事情。

  • 在中指中心线内侧弯曲

    精力略显不足，对外印象比较温和内敛，但对异性吸引力薄弱。

    多有小病，注意传染性疾病。如生命线起点越靠近中指，则此人越有勇气，做事积极。

  • 在中指中心线外侧弯曲

    此类人做事精力十足，不知疲倦，容易早恋。

    对情感的理解深于同龄人，爱情一般都能甜蜜。如果感情线紊乱，则此人多情，处理感情欠果断。

  • 出发点为弧形后呈直线

    这种人少年体力较好运气颇佳，但缺乏长期奋斗耐力。

    如果生命线靠近大拇指且智慧线较短，则容易冲动，难成大器。

  • 出发点为直线后呈弧形

    体质较弱，做事注意细节认真勤奋，容易神经紧张。

    如生命线后半段弧形延伸至手腕，表示中老年运会变好，老年生活安定。

  • 生命线末端分叉且弧线优美

    这种人对生活充满希望，健康长寿。

    生命线末端分叉，若支线紊乱，则容易沉溺在某种兴趣或理想中。

  • 生命线起点部分呈锁链状

    年幼体弱多病，生活按压能力差，容易感冒腹泻。

    起点部分呈锁链状，且中断出现朝命运线延伸的上升线：表示中年以后运势会上升，成功运相比好很多。

  • 生命线中间断裂

    断裂处通过流年法计算得出的年份，患病机率高。

    如断裂起点在生命线外侧，此年疾病不会太严重。如果断裂处出现岛纹或横线，要加强保健，提防交通事故。

  • 内侧出现第二条生命线

    生命力强盛能抵御疾病困扰，辅助生命线加强健康运。

    如辅助线有断裂，恢复能力较好，但感情易出问题。如果辅助线起点在生命线的中部靠下方，表示起运晚。
