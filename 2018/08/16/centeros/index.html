Centos 命令

简介

Linux 本质是指 Linux 内核，而称 GNU/Linux 则代表这是一个系统，所以我认为 Debian 的这个叫法是合理的，但是确实有点不好念和记忆。所以普遍大家直接称作 Linux。

  1. Fedora、RHEL、Centos 是一个系：业界一般用于做服务器
  2. Debian、Ubuntu 是一个系的：业界一般用于桌面，移动端，TV这一类
  3. ArchLinux 自己一个系：很轻量的Linux，适合有一定Linux基础，并且爱折腾的人使用，用它做桌面或是服务器都行
  4. openSUSE 自己一个系：人气相对比较差，一般是服务器
推荐的发行版本

Ubuntu：适用于开发机
推荐版本：Ubuntu kylin 15.10
CentOS：适用于服务器机
推荐版本：6.7

下载：wget 链接

创建文件夹：mkdir 文件夹名称

查看整台服务器的硬盘使用情况：df -h

查看每个文件夹的大小：du -sh *

输出当前路径：pwd

压缩文件或文件夹：zip -r zipName(如resources.zip) fileName(如resources或者resources.log)

解压文件：unzip zipName(如resources.zip)

Docker 命令

上线： docker-compose up -d

下线： docker-compose down

重启： docker-compose restart

显示所有的容器，包括未运行的： docker ps -a

进入指定容器： docker exec -it containerId bash

查看指定容器的日志：docker logs --since 1m containerId

Mysql命令

登录： msyql -u 用户名 -p;

显示数据库列表： show databases;

进入数据库： use mysql;

显示库中的数据表： show tables;
